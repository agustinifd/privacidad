Política de Privacidad

Última actualización: 23/12/2025

1. Información que recopilamos

Esta aplicación funciona de manera offline y no recopila ni transmite información personal. Todos los datos se almacenan únicamente en el dispositivo del usuario mediante una base de datos local (SQLite).

2. Uso de la información

Los datos guardados en el dispositivo se utilizan exclusivamente para el funcionamiento interno de la aplicación.

3. Compartición de datos

No compartimos información con terceros.

4. Seguridad

La seguridad de los datos depende del dispositivo del usuario. Se recomienda mantenerlo protegido y actualizado.

5. Privacidad de menores

La aplicación no recopila información personal de menores de edad.
